1
Objective
⌄

Trusterity requires a fully consent-safe cookie setup that is compliant with both GDPR (General Data Protection Regulation) and CCPA (California Consumer Privacy Act). The implementation blocks all analytics and ad tracking until a user explicitly opts in, and gives users the ability to withdraw or modify consent at any time.

HubSpot manages the consent banner, Google Consent Mode holds denied defaults from the moment of page load, and Google Tag Manager fires tracking tags only after consent is explicitly granted.

☑ GDPR requirements met
  • Consent denied by default on page load
  • Explicit opt-in before any tracking
  • Granular cookie category controls
  • Consent withdrawable at any time via footer link
  • No pre-ticked consent boxes
☑ CCPA requirements met
  • Users notified of data collection at first visit
  • Right to opt out honoured via Decline button
  • GPC / Global Privacy Control signal respected
  • No sale or sharing of personal data before consent
  • Cookie settings accessible from every page footer

Head code — Consent Mode + HubSpot bridge
⌄
Placement

Webflow → Project Settings → Custom Code → Head Code. Must appear before the GTM snippet.

html 
<!-- Consent Mode Default + HubSpot Consent Bridge -->
<script>
window.dataLayer = window.dataLayer || [];

function gtag() {
    window.dataLayer.push(arguments);
}

gtag('consent', 'default', {
    ad_storage: 'denied',
    analytics_storage: 'denied',
    ad_user_data: 'denied',
    ad_personalization: 'denied',
    functionality_storage: 'denied',
    personalization_storage: 'denied',
    security_storage: 'granted'
});

window._hsp = window._hsp || [];

window._hsp.push([
    'addPrivacyConsentListener',
    function (consent) {
        var hasConsent = !!(consent && consent.allowed);

        if (hasConsent) {
            gtag('consent', 'update', {
                ad_storage: 'granted',
                analytics_storage: 'granted',
                ad_user_data: 'granted',
                ad_personalization: 'granted',
                functionality_storage: 'granted',
                personalization_storage: 'granted',
                security_storage: 'granted'
            });

            if (!window.__hubspotConsentGrantedPushed) {
                window.__hubspotConsentGrantedPushed = true;

                window.dataLayer.push({
                    event: 'hubspot_consent_granted',
                    hubspotConsentGranted: true
                });
            }
        } else {
            window.__hubspotConsentGrantedPushed = false;

            gtag('consent', 'update', {
                ad_storage: 'denied',
                analytics_storage: 'denied',
                ad_user_data: 'denied',
                ad_personalization: 'denied',
                functionality_storage: 'denied',
                personalization_storage: 'denied',
                security_storage: 'granted'
            });

            window.dataLayer.push({
                event: 'hubspot_consent_denied',
                hubspotConsentGranted: false
            });
        }
    }
]);
</script>

Head code — Google Tag Manager
⌄
Placement

Add immediately after the Consent Mode block in Head. Container: GTM-WQWKKLH5

html 
<!-- Google Tag Manager -->
<script>
window.dataLayer = window.dataLayer || [];

(function(w,d,s,l,i){
    w[l]=w[l]||[];
    w[l].push({'gtm.start': new Date().getTime(), event:'gtm.js'});
    var f=d.getElementsByTagName(s)[0],
        j=d.createElement(s),
        dl=l!='dataLayer'?'&l='+l:'';

    j.async=true;
    j.src='https://www.googletagmanager.com/gtm.js?id='+i+dl;
    f.parentNode.insertBefore(j,f);
})(window,document,'script','dataLayer','GTM-WQWKKLH5');
</script>
<!-- End Google Tag Manager -->

Footer code — Banner reopen script
⌄
Placement

Webflow → Project Settings → Custom Code → Footer Code (before </body>)

html 
<style>
.cookie-settings-link {
    cursor: pointer;
    text-decoration: underline;
}
</style>

<script>
document.addEventListener('click', function (event) {
    var trigger = event.target.closest('[data-open-cookie-settings]');
    if (!trigger) return;

    event.preventDefault();

    var _hsp = window._hsp = window._hsp || [];
    _hsp.push(['showBanner']);
});
</script>

GTM configuration
⌄

All tracking tags must use a single custom event trigger. Never use the "All Pages" trigger for any analytics or ad pixel.

Required trigger

Custom Event: hubspot_consent_granted

Apply to these tags:

GA4 Meta Pixel Twitter/X Pixel LinkedIn Insight Tag Any future pixels
Do not use

All Pages trigger · inline fbq / twq / gtag scripts · tracking scripts inside Webflow Head

Validation checklist
⌄

Before accepting consent — Network tab should show no requests to:

collect facebook.com twitter.com linkedin.com analytics

After accepting:

  • event: hubspot_consent_granted appears in dataLayer
  • GTM tags fire in GTM Preview
  • GA4 + ad pixels begin sending events

After declining:

  • event: hubspot_consent_denied appears in dataLayer
  • Tracking remains fully blocked
